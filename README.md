# Новое русское вино

Сайт магазина авторского вина "Новое русское вино" с автоматическим заполнением данных по товарам из Excel-файла.

## Общие требования

1. Python3 должен быть уже установлен.
2. Скачайте код сайта.
3. Рекомендуется использовать [virtualenv/venv](https://docs.python.org/3/library/venv.html) для изоляции сайта. 
4. Затем используйте `pip` для установки зависимостей с помощью следующего кода:  
    ```  
    pip install -r requirements.txt
    ```
5. Потребуется Excel-файл с данными по товарам. В файле должны быть столбцы: 
- Категория (категория товара, например, Белые вина)
- Название (название товара, например, Киндзмараули)
- Сорт (сорт винограда - только для вина, например, Саперави)
- Цена (цена товара в рублях, например, 500)
- Картинка (полное название фото/картинки товара, расположенной в папке "images", например, kindzmarauli.png)
- Акция (Название акции, если товар участвует в ней, например, Выгодное предложение)
Одна строка - один товар.  
По умолчанию данный файл должен находиться в корневой папке сайта с названием "wine.xlsx".
6. Если Excel-файл удобнее хранить в другой папке или с другим названием, можно указать альтернативный путь (относительный или абсолютный) к файлу.   
    Для этого есть 2 способа.
    1. Указать путь в специальном файле ".env".  
        Для этого в корневой папке сайта создайте файл ".env".  
        В этом файле укажите путь к Excel-файлу в переменной `EXCEL_PATH`,
        Пример:
        ```
        EXCEL_PATH=file.xlsx
        ```
    2. При запуске сайта указывать путь к Excel-файлу как входной аргумент.  
        Например,  
        ```
        python .\main.py C:\my_files\file.xlsx
        ```

## Запуск

1. Когда все необходимые требования выполнены, в командной строке перейдите в папку сайта и запустите сайт командой: 
    ```
    python .\main.py [можно ввести альтернативный путь до Excel-файла с данными о товарах]
    ```  
2. Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

### Примеры запуска приложения 

1. Запуск сайта без указания альтернативного пути к Excel-файлу.  
    ```
    python .\main.py 
    ```
2. Запуск сайта с указанием альтернативного пути к Excel-файлу.  
    ```
    python .\main.py C:\my_files\file.xlsx
    ```
